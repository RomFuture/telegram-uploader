Telegram Uploader — one-time sign-in
====================================

Why sign in?
  Backups are uploaded with YOUR Telegram account (Client API), not a bot.
  Telegram asks for your phone number and a login code — same as the official app.

Your data stays on this computer:
  • Session file — only on your disk (see Settings → Session file path)
  • API id/hash — in ~/.config/telegram-uploader/.env
  • Backups go only to the private group ID you configure
  • Phone, codes, and passwords go to Telegram servers only — not to us

Before sign-in:
  1. Open the app → Settings → fill API ID, API hash (my.telegram.org), group ID
  2. Click Save (writes ~/.config/telegram-uploader/.env)
  3. Run this login step once

You may be asked for:
  • Phone number (international format, e.g. +79001234567)
  • Code from Telegram app
  • 2FA password (if enabled on your account)

Press Enter to open the login prompt, or Ctrl+C to cancel.
